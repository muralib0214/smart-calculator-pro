
# 🚀 Smart Calculator Pro

### A Professional Python-Based CLI Calculator with Persistent History & Smart Error Handling

---

# 📖 Project Overview

Smart Calculator Pro is a professional command-line calculator application built using Python.

The project is designed to demonstrate strong programming fundamentals, clean code structure, exception handling, file management, and user-friendly CLI application development.

It performs essential arithmetic operations while maintaining a persistent calculation history with timestamps.

---

# ✨ Features

✅ Addition  
✅ Subtraction  
✅ Multiplication  
✅ Division  
✅ Modulus Operation  
✅ Smart Input Validation  
✅ Division by Zero Protection  
✅ Persistent History Storage  
✅ Timestamped Calculations  
✅ Clear History Feature  
✅ Professional Command-Line Interface  
✅ Stores Last 10 Calculations

Smart Calculator Pro handles:

* Invalid numeric inputs
* Division by zero
* Modulus by zero
* Invalid menu selections
* Missing history files

This ensures a smooth and reliable user experience.
